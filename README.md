# 构建说明

基于github的github-page
使用开源项目[hexo](https://hexo.io)构建托管于`github`上的静态博客
主题采用的是@iissnan贡献的[hexo-theme-next](https://github.com/iissnan/hexo-theme-next)
本人在有些地方做了一些修改，具体修改请见下列条目
